commands to run  for python :-
1. upgrading pip :python.exe -m pip install --upgrade pip
2. starting the virtual environment :  
	step1 : python -m venv nameofenvironment(python -m venv env)
	step2: env\Scripts\activate
	step3: to deactivate the virtual environment : deactivate
3. installing any package : pip install package-name 
4. running a fast api server using uvicorn : 
    step1: pip install "uvicorn[standard]"
    step2: uvicorn main:app --reload --port 9999

    also in fast api - even if all the function  is synchrnonpous and there is no  use of async anywhere it 
    will still be synchronous as it runs on starlet
